# Dress Decode Project

A fun fashion-focused web project built to explore website design, styling, and user interaction.

This project was created as a creative experiment and an opportunity to practice front-end web development skills.

## Built With

* HTML
* CSS
* JavaScript

## Features

* Responsive design
* Interactive user experience
* Fashion-inspired interface
* Mobile-friendly layout

## About

Dress Decode was built as a personal project to experiment with web design and create something fun around fashion and style.
